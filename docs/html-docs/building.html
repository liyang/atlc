Building atlc from sources

atlc and most of its assocoiated programs have been successfully build on the following platforms.
  1. Dual processor Sun Ultra 60 running Solaris 8 with the 108528-12 kernel
  2. Quad processor (4x450 MHz) Sun Ultra 80 running Solaris 9
  3. 64-bit Dec Alpha, running Linux 2.4.17
  4. Sun Ultra 60 running Solaris 8 with the 108528-11 kernel
  5. x86 PC running linux 2.4.13
  6. Sun (unknown) running Linux 2.2.18pre21
  7. Dual processor DEC Alpha running Linux 2.4.17
  8. Sun R220 running Solaris 8
  9. x86 PC's running Windows 98 and NT
Download atlc from atlc's download area on SourceForge. It is preferable to also have following, although none, apart from a compatible version of make, are essential.
  1. If you have a system with more than one CPU, atlc can be configured to use any number of CPUs in parallel. You will need to install pthreads, or other POSIX/ANSI-C multi-threaded library. Sun's commerical implementation works fine, but I'm unsure about other versions.
  2. The GNU scientific library, gsl.
  3. GNU make, sometimes installed as gmake. Sun's version of make in /usr/ccs/bin/make is not suitable, although I don't know about other versions of make

Extract the compressed file, using gunzip, or gzip -d. 
% gzip -d atlc-X.Y.tar.gz
% tar xvf atlc-X.Y.tar
% cd atlc-X.Y
% ./configure
% make 
% make check   // This is very important, as it cheks and also builds the examples
% su
# make install.
If you have a system with more than one CPU, atlc can be configured to use any number of cpus in parallel. You will need to install pthreads, portable POSIX/ANSI-C based multi-threaded library for Unix platforms. Installation is identical to above, except that the option --with-threads should be passed to configure i.e. 
% configure --with-threads
For a list of configure's options, type 
% configure --help

Note, altc should first be configured without threads and make check run. The algorithm used for a threaded version of atlc is slightly different, but will result in a lot of tests failinig, as the files are slightly different and so have different checksums. a
